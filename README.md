## ⭐ Proyectos Destacados

### [TaskFlow](https://github.com/miguel3912/taskflow)
🚀 Aplicación de gestión de tareas colaborativa con React, Node.js y MongoDB. Incluye autenticación JWT, sincronización en tiempo real con WebSockets y panel de análisis avanzado.

**Tech**: React • Node.js • MongoDB • Socket.io

### [EcommercePro](https://github.com/miguel3912/ecommerce-pro)
🛍️ Plataforma de e-commerce completa con carrito de compras, sistema de pagos integrado con Stripe, panel de administración y dashboard de ventas.

**Tech**: Next.js • Express • PostgreSQL • Stripe API

### [WeatherApp](https://github.com/miguel3912/weather-app)
🌤️ Aplicación de clima moderna con geolocalización, predicción de 5 días y tema oscuro/claro. Implementada con React Hooks y OpenWeather API.

**Tech**: React • OpenWeather API • Tailwind CSS

### [CodeSnippets](https://github.com/miguel3912/code-snippets)
📝 Gestor de fragmentos de código colaborativo para desarrolladores. Permite compartir, buscar y organizar snippets con tags.

**Tech**: React • Firebase • Algolia Search
